# Whatsapp-Chat-Contribution-Analyzer
Script made for fun to analyze exported WhatsApp chats and find out who sends the most messages in a group.

## How to use ?
1. Open a 'WhatsApp' group chat
2. Open (⋮) then 'Export chat'
3. Choose Without media
4. Clone this project 
```bash
git clone https://github.com/Kraids99/Whatsapp-Chat-Contribution-Analyzer.git
```
5. Put your file inside the data folder, then run the script
```
Whatsapp-Chat-Contribution-Analyzer/
├─ data/
│  └─ chat.txt
└─ analyzer.js
```

Note: You can rename chat.txt to anything, just make sure inside the data folder.
